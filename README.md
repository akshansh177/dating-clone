Project is divided into two parts:\
1. Frontend (client)\
2. Backend (server) ~ port:8000

For Frontend:

```bat
cd client
npm i
npm install -g react-scripts
react-scripts start
```

For Backend(using 'Nodemon'):

```bash
cd server
npm i
npm install -g nodemon
nodemon index.js
```
